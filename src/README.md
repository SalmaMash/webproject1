# Project 1 — Component-Based Front-End Data Display


This project is a React SPA demonstrating client-side routing, mock data loaded into state, filtering and sorting using native array methods, responsive UI, and placeholder auth pages.


## How to run locally
1. Create a new folder and copy the files as shown.
2. Ensure Node.js (>=16) is installed.
3. Run:


```bash
npm install
npm run dev
```


Open the app on the address printed by Vite (usually http://localhost:5173).

## Deploy to Render (Static Site)
1. Push repository to GitHub.
2. Create a new Static Site on Render.com and connect your GitHub repo.
3. Set the build command to: `npm run build`
4. Set the publish directory to: `dist`
5. Start the deploy — Render will build and give you a live URL.


Remember to include the Render URL in your submission.
